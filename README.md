GlamGenius is a human-centered AI wardrobe and outfit recommendation system.  
The application helps users manage clothing items, generate outfit combinations, and plan outfits for specific dates.

The project focuses on combining practical software engineering with AI-ready recommendation logic and clean full-stack architecture.

---

## Features

- Wardrobe management (add, edit, delete clothing items)
- Clothing categorization (top, bottom, shoes, outerwear, accessories)
- Tag and season-based filtering
- Rule-based outfit generation (MVP)
- Outfit saving and reuse
- Calendar-based outfit planning
- Structured data models for future ML-based recommendations

---

## Tech Stack

Frontend:
- Next.js (App Router)
- TypeScript
- Tailwind CSS

Backend:
- Next.js API routes
- Prisma ORM

Database:
- SQLite (local dev)

Validation & Structure:
- Zod validation
- Typed data models
- Modular component architecture

This project was built to demonstrate:

- Recommendation logic design
- Human-centered AI product thinking
- Scalable architecture for future ML integration
- Production-style project structure
